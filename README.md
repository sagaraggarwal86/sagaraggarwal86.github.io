# sagaraggarwal86.github.io

Personal portfolio site — landing page for my JMeter plugins.

Live at: <https://sagaraggarwal86.github.io>

## Repo layout

```
.
├── index.html          # the whole site
├── assets/
│   └── sagar.jpg       # profile photo (add yours here)
└── README.md
```

## To update

1. Edit `index.html` directly (it's a single file — no build step).
2. Commit and push to `main`.
3. GitHub Pages rebuilds automatically (~1 minute).

## To add the photo

Drop a square photo (400×400 minimum, ≤200 KB) at `assets/sagar.jpg`. Until then, the site shows initials as a fallback.
